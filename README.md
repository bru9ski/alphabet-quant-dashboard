# AlphaBet Quant Dashboard

Quantitative Betting Intelligence dashboard com tema **Dark + Yellow**.

## Features
- Motor de Poisson & xG
- Value Bet Detector (EV > +5%)
- Construtor de Bilhete @2.00
- Kelly Fracionário ¼
- **LocalStorage** — salva banca, inputs e bilhete automaticamente

## Stack
- HTML + TailwindCSS (CDN)
- JavaScript vanilla
- Lucide Icons

## Como usar
Abra o `index.html` no navegador. Nenhuma dependência para instalar.
